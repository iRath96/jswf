Detailed Description

Represents CXFORM and CXFORMWITHALPHA records.

For each channel \(v\) from \({r,g,b,a}\), the transformed value \(v'\) can be computed as follows:
\(v' = max(0, min((v \cdot vM \cdot \frac{1}{256}) + vA, 255))\)
