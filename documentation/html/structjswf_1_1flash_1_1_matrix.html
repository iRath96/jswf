bool jswf::flash::Matrix::inverse ( Matrix &  result) const
inline

Calculates the inverse of this Matrix.

\[ \begin{bmatrix} s_x & r_1 & t_x \\ r_0 & s_y & t_y \\ 0 & 0 & 1 \end{bmatrix}^{-1} = \frac{1}{s_x s_y - r_0 r_1} \begin{bmatrix} s_y & -r_1 & t_y r_1 - t_x s_y \\ -r_0 & s_x & t_x r_0 - t_y s_x \\ 0 & 0 & 1 \end{bmatrix} \]

Parameters
[out]resultA reference to the Matrix to store the result in.
Returns
true if the operation was successful, false if the Matrix cannot be inversed
Warning
This will fail if the determinant of this Matrix ( \(s_x s_y - r_0 r_1\)) equals zero. In that case the given Matrix cannot be inversed and no write to result occurs.

Calculates the product of two instances of Matrix.

given \(a, b, c \in \mathbb{R}^{3 \times 3}\)

\[a \left( b \begin{pmatrix}x\\y\\1\end{pmatrix} \right) = c \begin{pmatrix}x\\y\\1\end{pmatrix}\]


\[ c = a \times b = \begin{bmatrix} s_{x_a}s_{x_b} + r_{1_a}r_{0_b} & s_{x_a}r_{1_b} + r_{1_a}s_{y_b} & s_{x_a}t_{x_b} + r_{1_a}t_{y_b} + t_{x_a} \\ s_{y_a}r_{0_b} + r_{0_a}s_{x_b} & s_{y_a}s_{y_b} + r_{0_a}r_{1_b} & s_{y_a}t_{y_b} + r_{0_a}t_{x_b} + t_{y_a} \\ 0 & 0 & 1 \end{bmatrix} \]
